# Lunari Frontend - Batch 4 Desktop Responsive

This package is based on the previous batch 4 code and adds responsive desktop behavior while keeping the mobile layout intact.

## Updated behavior
- Mobile layout still works as before
- Desktop now shows a wider app shell
- Primary screens adapt to multi-column layouts on large screens
- Bottom navigation stays usable across screen sizes
- Home, calendar, add-log, tracker, and settings are improved for desktop use

## Run
npm install
npm run dev

Open http://localhost:3000
